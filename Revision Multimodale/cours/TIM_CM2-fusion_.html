<h3>Multimodal fusion</h3>

<div class="r-stretch">
<p style="text-decoration: underline;">Early and late fusion</p>

<ul>
<li><strong>Feature / recognition-based / early fusion</strong>: each input modality is viewed as a feature input to a "model" (e.g., HMM, statistic, ANN) that integrates and interprets it</li>
<li><strong>Semantic / decision-based / late fusion</strong>: using a dictionary-like technique to reach a complete interpretation</li>
</ul>

</div>

</ul>
